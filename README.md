#Underground Dash
OS X Catalina removes the dashboard feature, bring it back with Underground Dash.

![alt text](./screenshots/screenshot-1.png "Image1")

## Features
- Calendars
- Sticky notes
- Clocks
- Weather
- RSS Feed

## Technical
- Electron
- Rivets.js

## Development
```
npm start
```
